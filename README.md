# Finance-Analyst-Project
# Financial Analysis & Portfolio Optimization Model

This project analyzes multiple global equities using Python to evaluate:

- Stock performance trends
- Risk vs return analysis
- Sharpe ratio comparison
- Correlation between assets
- Portfolio optimization using Monte Carlo simulation

## Key Outcome
Developed an optimal portfolio allocation based on maximum Sharpe Ratio to achieve the best risk-adjusted returns.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Google Colab

## Companies Analyzed
Apple, Microsoft, Google, Meta, NVIDIA, Oracle, Tesla, TCS, Reliance
